Inspector Bota

Discord PUBG Scrim Bot

What is Inspector Bota?

A Discord bot to manage PUBG scrim matches, teams, check-ins, lootspot lobbies, and staff workflows. Built for server owners, staff, and players, it creates required channels and roles, then lets registered teams and mix teams participate in a PUBG-style scrim flow.

What it does

  • Manages PUBG scrim match setup and team check-ins
  • Creates channels and roles automatically with /init
  • Supports private staff commands and public player commands
  • Generates lobby-based lootspot selection channels
  • Handles team check-in, checkout, voice channel creation, and matchmaking priority

Command Types

There are two command groups:

  • Public (Open): for all server members
  • Private: for staff members only

To start, the server owner must run /init. This creates required channels and roles. After initialization, private commands must be used in the 📑│bot-commands channel.

Scrim Commands

  • /init — Initialize the bot (server owner only)
  • /scrim — Open, close, reset, announce scrim
  • /scrim_edit — Change min/max teams and mode

Lootspot Commands

  • /ls_create — Create lobby category and channels
  • /ls_active — Send lootspot options
  • /ls_lock — Lock/unlock lootspots

Team Commands

  • /add — Register a team
  • /list — List registered teams
  • /checkin — Check in registered team
  • /checkin_mix — Check in mix team
  • /checkout — Checkout team

Usage Guide

Once /init, is executed, the bot will create the required roles, channels, including check-in, check-out, and bot-commands.

Staff can run private commands only in the bot-commands channel. When teams use the /checkin command, voice channels are created automatically. The team lead can also add temporary players to the channel.

A set of lobbies and loot spot options has been created with /ls_create, allowing teams to make their selections.
